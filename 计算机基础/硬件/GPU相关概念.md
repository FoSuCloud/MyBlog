## 超频
超频（`Overclocking`）是一种技术，指的是将计算机硬件组件（如中央处理器 CPU、图形处理器 GPU、内存 RAM 等）的
`时钟频率`或`工作电压`调整到`高于制造商指定的标准值`，`以获得更高的性能`。

在默认情况下，硬件组件的时钟频率和电压通常`被设置为安全和稳定的工作范围`，以确保它们在各种条件下可靠地运行。
然而，一些电脑发烧友和玩家希望通过超频来提升硬件的性能，以获得更快的处理速度、更高的图形渲染性能或更大的计算能力。

超频的过程通常包括：
`时钟频率调整`： 增加硬件组件的时钟频率，如CPU或GPU的主频，以使其运行更快。这可以提高处理速度和计算性能，但可能会导致温度升高和功耗增加。
`电压调整`： 在某些情况下，增加硬件组件的工作电压可以帮助稳定超频过程，但也会导致功耗和发热增加。调整电压需要谨慎，以避免损坏硬件。
`稳定性测试`： 超频后，通常需要进行稳定性测试，以确保硬件在高频率下能够稳定工作，不会引发崩溃或错误。
`温度监控`： 超频可能导致硬件温度升高，因此需要定期监控温度，以避免过热损坏硬件。

超频可以显著提高计算机的性能，但也存在一些潜在风险。`不正确的超频可能会导致系统崩溃`、`硬件损坏或不稳定的操作`。
因此，在进行超频之前，用户需要了解硬件规格、性能潜力和超频过程，以及采取适当的预防措施。

#### Video Memory显存
* 显存（Video Memory，也称为显存储器或显存储存器）是一种`专门用于存储图形数据`的高速内存，
* 位于图形处理器（GPU）上或者与GPU紧密集成的显卡上。
* 显存用于存储图像、纹理、帧缓冲、着色器程序和其他图形渲染所需的数据。

#### Memory Bus Width显存位宽
* 显存位宽（Memory Bus Width）： 显存位宽是`显存数据总线的宽度`，以位（bit）为单位。
* 它决定了`一次性可以传输多少位（或字节）的数据`。较大的位宽通常意味着更大的数据吞吐量和更快的数据传输速度。
* 例如，256位宽的显存可以在每个时钟周期传输更多的数据，比128位宽的显存更快。

### Memory Bandwidth显存带宽
* 显存带宽（Memory Bandwidth）： 显存带宽是指`显存数据总线每秒传输的数据量`，通常以`GB/s`（千兆字节每秒）为单位。
* 它是`显存位宽、显存时钟频率和数据传输协议等因素的综合反映`。
* 较高的显存带宽意味着GPU可以更快地读写显存，从而提高图形渲染的性能。

#### CUDA数量
* GPU的CUDA(`Compute Unified Device Architecture 计算统一设备架构`)数量通常指的是该`GPU支持的CUDA核心数目`，
* `即其计算单元的数量`。这些CUDA核心用于执行并行计算任务，包括通用计算、图像处理、科学计算等。
