## pip 安装python项目依赖
* 如果两个项目之间存在依赖关系，那么使用的命令是
`pip install -e .`
  
## 安装requirement.txt
* pip install -r requirement.txt

#### 安装某个库指定pip源
* 可以在使用pip的时候在后面加上-i参数，指定pip源
* `pip install scrapy -i https://pypi.tuna.tsinghua.edu.cn/simple`

* `https://pypi.org/ 官方的源`
