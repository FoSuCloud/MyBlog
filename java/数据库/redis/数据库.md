## 数据库
* Redis支持多个数据库，用于在同一个Redis实例中存储不同的数据集。
* `每个数据库都由一个数字标识，从0开始递增。默认情况下，Redis实例中有16个数据库（编号从0到15）`

#### 查询数据库
* `CONFIG GET databases`
* 该命令将返回一个数组，其中第二个元素是Redis服务器配置中设置的数据库数量。
* 返回值如下所示：
```redis
1) "databases"
2) "16"
```
* "databases"是配置项的名称，它指示下一个键值对是关于Redis数据库数量的信息。
* "16"表示Redis服务器配置中设置的数据库数量。在默认情况下，Redis有16个数据库（从0到15）可供使用。
* `可以通过修改Redis的配置文件（redis.conf）或者在启动时使用参数来调整数据库的数量`。

#### 切换到数据库
* `SELECT 1 (这里的1就是db1)`

#### 显示数据库中的所有key
* `keys *`
* `该命令将返回选中的数据库中的所有键作为列表。`
