### 包装类
* `包装类都是不可变类。所谓不可变是指实例对象一旦创建，就没有办法修改了。`
* 这是通过如下方式强制实现的：
* ❑ 所有包装类都声明为了final，不能被继承。
* ❑ 内部基本类型值是私有的，且声明为了final。
* ❑ 没有定义setter方法。
* 为什么要定义为不可变类呢？`不可变使得程序更为简单安全，因为不用操心数据被意外改写的可能，`
* 可以安全地共享数据，尤其是在多线程的环境下。
