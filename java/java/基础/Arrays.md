## Arrays

### int array
* int数组初始化例子
```text
int[] primes = {2, 3, 5, 7, 11, 13, 17};
```
* `注意需要使用{}花括号，而不是[]`

### Integer array
```text
Integer[] primes = new Integer[]{2, 3, 5, 7, 11, 13, 17};
```
* `语法是new Integer[]{};`

### String array
* `初始化的时候定义String数组长度，后面再根据索引赋值`
```text
      String[] weekDays = new String[7];
      weekDays[0] = "Monday ";
      weekDays[1] = "Tuesday ";
      weekDays[2] = "Wednesday ";
      weekDays[3] = "Thursday ";
      weekDays[4] = "Friday ";
      weekDays[5] = "Saturday ";
      weekDays[6] = "Sunday ";
```
* `初始化的时候就赋值`
```text
public static final String[] USER_HEADER = new String[]{"用户名", "昵称", "手机号"};
```
